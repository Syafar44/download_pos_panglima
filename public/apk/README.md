# Folder APK

Folder ini digunakan untuk menyimpan file APK aplikasi POS Panglima yang dapat didownload.

## 📝 Instruksi

1. **Letakkan file APK di folder ini**
   - File naming: `pos-panglima-v[VERSION].apk`
   - Contoh: `pos-panglima-v1.0.0.apk`

2. **Update konfigurasi di `lib/app-versions.ts`**
   - Sesuaikan `filename` dengan nama file APK
   - Sesuaikan versi dan changelog
   - Set `isLatest: true` untuk versi terbaru

3. **Verifikasi**
   - Pastikan link download berfungsi
   - Test di browser

## 📦 Versi yang Tersedia

- `pos-panglima-v1.0.0.apk` (45 MB)
- `pos-panglima-v1.1.0.apk` (48 MB)
- `pos-panglima-v1.2.0.apk` (52 MB)

---

**Catatan**: File APK perlu ditambahkan secara manual ke folder ini. Pastikan file sudah tersedia sebelum di-deploy.
