⚡ QUICK START
================================================================================

1. Install dependencies:
   $ pnpm install

2. Run development server:
   $ pnpm dev

3. Open browser:
   http://localhost:3000

4. Edit to customize:
   lib/app-versions.ts    - Update versions
   lib/guide-data.ts      - Update panduan
   app/globals.css        - Change colors

5. Deploy:
   git push origin main
   (Vercel auto-deploys)

🎯 COMMON TASKS
================================================================================

Add new version:
  1. Copy APK → public/apk/pos-panglima-v1.3.0.apk
  2. Edit lib/app-versions.ts
  3. Test: pnpm dev
  4. Deploy: git push

Update guide:
  1. Edit lib/guide-data.ts
  2. Test: pnpm dev
  3. Deploy: git push

Change colors:
  1. Edit app/globals.css
  2. Auto-reload in browser
  3. Deploy: git push

Before deploy:
  1. Open QA_CHECKLIST.md
  2. Run through checklist
  3. Fix any issues
  4. Deploy: git push
